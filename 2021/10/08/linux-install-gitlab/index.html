linux 安装 gitlab

取得镜像源

首先 yum 是可以直接下载 gitlab-se 的， 但是好像很慢， 就打算把它下载到本地之后在通过本地安装

之前

yum install gitabxxxxxx.rpm

现在是我们直接吧 rpm文件下载好，然后在上传到服务器上 ， 这里推荐腾讯云 ， 不推荐使用 清华的 ， 清华的也很慢 不确定是不是我的网速问题

安装

yum localinstall ./gitlab-12.0.0.stable.rpm

在 run 之前，我们需要更改一下配置 ，就是gitlab 的 IP地址 ， 由于我这里是虚拟机 ， 用的就是 127.0.0.1:10000 (如果不带端口，就是80 ， 可能会冲突)

之后我们 , 重新饮用配置

gitlab-ctl reconfigure

再进行运行

run

gitlab-ctl start

当使用了 gitlab start 那么，这个gitlab 就已经开启了， 并且会开启很多很多服务 ， 之后我们就可以去访问了

stop

gitlab-ctl stop

restart

gitlab-ctl restart

查看各服务的状态

gitlab-ctl status

汉化

其实汉化这个功能我没有弄出来一个完美的方案，也在网上参考了一些文章 ， 直接说一下结果吧。。。

  1. 先去拉取 gitlab 的中文包
  2. 然后再拷贝到 /opt/gitlab/embedded/service/gitlab-rails
  3. reconfigure ， restart

我自己尝试出来是部分汉化， 并且很多功能直接报错 500 ， 然后就没有尝试其他的了 ， 其实还有一种方法是通过git , 其实英文版也能用

算了
